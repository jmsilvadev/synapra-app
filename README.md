# synapra-app

Aplicacao React para clientes Synapra gerenciarem plano, consumo, faturamento e add-ons.

Esta app usa o mesmo runtime base do `../surf-frontend`: `react-scripts` + `react-app-rewired` + TypeScript.

## Rodar localmente

```bash
npm install
npm start
```

## Build

```bash
npm run build
```

## Escopo atual

- Dashboard comercial com resumo de conta
- Gestao de plano com troca visual entre tiers
- Visualizacao de uso do ciclo atual
- Historico de faturas e add-ons
- Dados mockados prontos para substituicao por API
